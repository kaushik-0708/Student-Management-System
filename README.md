# 🎓 Student Management System

A simple and interactive **Student Management System** developed using **Python** and **Streamlit**. This project allows users to manage student records digitally with features like adding, viewing, searching, and deleting student data.

---

# 🚀 Features

* 🔐 Login Authentication
* ➕ Add Student Records
* 📋 View All Students
* 🔍 Search Student by Roll Number
* 🗑 Delete Student Records
* 📊 Automatic Grade Calculation
* 💾 JSON File Data Storage
* 📥 Download Student Data as CSV
* 🎨 Interactive Streamlit UI

---

# 🛠 Technologies Used

* **Python** → Backend Logic
* **Streamlit** → User Interface
* **JSON** → Data Storage
* **Pandas** → Data Handling & CSV Export
* **Object-Oriented Programming (OOP)** → Project Structure
* **VS Code** → Development Environment

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/student-management-system.git
```

## 2️⃣ Open Project Folder

```bash
cd student-management-system
```

## 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

Use the following command:

```bash
streamlit run app.py
```

The application will automatically open in your browser.
